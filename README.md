# 🥒 Pickle (피클)
> **"방대한 정보의 껍질은 벗기고, 알맹이만 아삭하게 절여낸 AI 요약 한 조각"**

"우리 요새 긴 글 잘 못 읽잖아요." <br> Pickle은 브라우저에서 마주하는 복잡한 정보를 핵심만 쏙 뽑아 정리해주는 AI 기반 요약 서비스입니다. <br>특히 영어 콘텐츠를 한국어로 먼저 요약해 읽음으로써 원문 이해도를 극대화하는 영어 공부의 치트키 역할을 합니다.

<br>

## ✨ 왜 피클(Pickle)인가요?
- **Crunchy Summary**: 아삭한 식감처럼 군더더기 없이 깔끔한 핵심 요약을 제공
- **Easy Pick**: Chrome 등 브라우저 확장 기능을 통해 단 두 번의 클릭으로 요약 완료.
- **Fresh Insight**: 글로벌 트렌드와 외국어 콘텐츠를 한국어 요약으로 먼저 전달하여 신선한 통찰 제공
- **Simple Design**: 누구나 쉽게 사용할 수 있는 직관적이고 간단한 인터페이스 지향
