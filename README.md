# 🥒 Pickle (피클) 
> **"방대한 정보의 껍질은 벗기고, 알맹이만 아삭하게 절여낸 AI 요약 한 조각"**

## 📖프로젝트 소개
길고 지루한 텍스트, 읽기 힘든 외국어 콘텐츠... 이제는 **Pickle** 하세요! <br>
Pickle은 사용자가 웹에서 마주하는 복잡한 정보를 핵심만 쏙 뽑아(Pick), 보기 좋게 정리해주는 AI 기반 요약 어플리케이션입니다.

## ✨Pickle's Points
- **Crunchy Summary:** 아삭한 식감처럼 군더더기 없이 깔끔한 핵심 요약
- **Easy Pick:** chrome 브라우저 공유 기능을 통해 단 두 번의 클릭으로 요약 완료
- **Fresh Insight:** 글로벌 트렌드와 현재 반응까지 한국어로 신선하게 전달
